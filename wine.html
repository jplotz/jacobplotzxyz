Regressions

Regressions are issues with code that come about as a result of changes. They are unexpected problems stemming from new updates to the codebase.

Wine uses Conformance Tests, which can sometimes catch regressions.

https://wiki.winehq.org/Regression_Testing

Basic Usage

Wine can be built and run in the same directory that contains the source code. Wine comes with the wine notepad command to test if the program works.

To run it, point it to a .exe file. It is also possible to use wine start [full-file-path-here].
